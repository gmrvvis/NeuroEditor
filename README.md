ReTracer
=====================================================

## Introduction

Editor of neuromorphological tracings.

## Dependencies

* Required dependencies for ReTracer:
    * NSOL
    * glm
    * libQGLViewer

## Building

Description

```bash
git clone --recursive https://gitlab.gmrv.es/gmrv/ReTracer
mkdir ReTracer/build && cd ReTracer/build
cmake .. -DCLONE_SUBPROJECTS=ON
make
```
